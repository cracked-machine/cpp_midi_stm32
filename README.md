## cpp_midi_stm32

Library for sending MIDI messages over USART

See the [wiki](https://github.com/cracked-machine/cpp_midi_stm32/wiki) for documentation / reference

See [readme](tests) for information on unit testing/mocking.


#### Adding this library to your STM32 Project

Add this repo ([cpp_midi_stm32](https://github.com/cracked-machine/cpp_midi_stm32.git)) to your project as a submodule and add the subdirectory:

```
add_subdirectory(cpp_midi_stm32)
```

[Embedded Utils](https://github.com/cracked-machine/embedded_utils)
